# 2. 유저 페르소나

---

## ② User Persona

MVP는 특정 인구통계보다 **"화면을 볼 수 없는 상황"**을 공유하는 사용자들을 타겟으로 합니다. 아래 3명은 실제 초기 사용자를 대표하는 원형(archetype)입니다.

---

### Persona 1. "몰래 확인러" — 직장인 축구팬

**이름 (가칭)**: 김민준, 34세

**직업**: IT 회사 대리, 사무직

**디바이스**: iPhone + Apple Watch (업무용으로 착용 중)

#### 상황

- 평일 오후 회의나 업무 중에도 유럽 리그 경기(예: EPL) 결과가 궁금함
- 폰을 자주 꺼낼 수 없는 환경 (회의 중, 상사 근처)
- 화장실이나 잠깐의 틈에만 몰래 스코어를 확인 가능

#### Pain Point

- 기존 앱은 골 알림이 와도 결국 폰을 켜서 확인해야 함 → 티가 남
- 워치를 봐도 "골! 손흥민 65분"처럼 텍스트라 결국 눈으로 읽어야 함
- 알림이 계속 울리면 옆 사람이 눈치챔 (소리/화면 밝기)

#### 원하는 것

- 손목이 진동하는 순간, **보지 않고도** "어, 골 들어갔구나" 를 직감적으로 알고 싶음
- 화면을 켜지 않고 자연스럽게 손을 무릎에 얹은 채로 정보 습득

#### 핵심 사용 시나리오

회의 중 손목에 짧고 강한 특정 패턴이 느껴짐 → 곁눈질도 없이 "아, 우리 팀 골 넣었네" 확신 → 회의 끝나고 확인.

### Persona 2. "이동 중 팬" — 통근/통학 축구팬

**이름 (가칭)**: 박서연, 27세

**직업**: 대학원생, 지하철로 편도 50분 통학

**디바이스**: iPhone (워치 없음, 초기엔 폰 진동만 사용)

#### 상황

- 지하철 안에서 이어폰으로 라디오 중계를 듣기엔 신호가 끊기거나 사람들 시선이 신경 쓰임
- 손잡이를 잡고 서서 가는 경우가 많아 폰을 계속 볼 수 없음
- 데이터도 아끼고 싶어서 무거운 앱 실행은 피하고 싶음

#### Pain Point

- 지하철에서 폰을 꺼내 화면을 계속 켜두는 게 불편하고 배터리 소모도 큼
- 이어폰으로 소리를 들으려 해도 지하 구간에서 스트리밍이 끊김
- "지금 위험한 상황인지, 그냥 평범한 흐름인지" 조차 알 수 없어 답답함

#### 원하는 것

- 주머니 속 폰의 진동만으로 "어, 지금 위험한 공격 상황이구나" 를 느끼고 싶음
- 경기 흐름의 긴장감 자체를 놓치고 싶지 않음 (단순 결과 알림이 아니라 "흐름")

#### 핵심 사용 시나리오

주머니 속 폰이 "촤르르륵" 패턴으로 진동 → 위험한 공격 상황임을 직감 → 잠깐 폰을 꺼내 확인하고 다시 집어넣음.

### Persona 3. "워치 얼리어답터" — Apple Watch 헤비 유저

**이름 (가칭)**: 이도현, 31세

**직업**: 프리랜서 디자이너, 재택 + 외부 미팅 혼합

**디바이스**: Apple Watch Series 최신 모델 (거의 항상 착용), iPhone은 가방 속

#### 상황

- 손목에 워치를 차고 있지만 폰은 가방이나 다른 방에 있는 경우가 많음
- 이미 다양한 워치 앱(피트니스, 알림 관리)을 적극적으로 사용 중 → 워치 중심 라이프스타일
- 축구를 좋아하지만 "경기 내내 폰을 붙잡고 있는 것"은 피곤하다고 느낌

#### Pain Point

- 대부분의 축구 앱은 워치 지원이 부실하거나, 워치에서도 결국 화면을 봐야 함
- 워치의 장점(항상 몸에 붙어있음)을 제대로 활용하는 축구 앱이 없음
- 알림이 너무 잦으면 워치 배터리와 손목 피로도 신경 쓰임

#### 원하는 것

- 워치 하나만으로 경기의 굵직한 흐름(골, 카드, 하프타임, 종료)을 완결적으로 파악
- 진동 패턴을 스스로 커스터마이징해서 "내 취향의 리듬"으로 경기를 느끼고 싶음

#### 핵심 사용 시나리오

카페에서 작업 중, 워치가 짧게 두 번 울림 → "옐로카드구나" 바로 인지, 작업 흐름 안 끊김 → 나중에 하프타임 패턴이 오면 자연스럽게 잠깐 쉬는 타이밍으로 활용.
